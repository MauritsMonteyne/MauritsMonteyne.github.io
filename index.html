Terminal 
                Maurits Monteyne

Welkom op mijn e-portfolio website.

Zoals je kan zien is dit geen gewone website. Gezien ik een groot deel van mijn tijd achter computer terminals doorbreng, kreeg ik het idee om van deze website ook een interactieve terminal te maken. Dit houdt in dat je onderdaan een commando moet intypen en er vervolgens een antwoord zal verschijnen.

Typ 'help' om een lijst met beschikbare commando's weer te geven.
